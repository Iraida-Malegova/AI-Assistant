# Проект: ИИ-ассистент для салона "Белар" по установке окон

## Описание проекта
Проект направлен на создание интеллектуального ассистента для салона "Белар", специализирующегося на установке окон. ИИ-ассистент разработан для автоматизации взаимодействия с клиентами, улучшения качества обслуживания и оптимизации процессов записи на встречи. Ассистент способен вести осмысленные диалоги, отвечать на вопросы клиентов на основе базы знаний, а также организовывать встречи и добавлять их в Google Календарь.

---

## Функционал

### 1. Осмысленное ведение диалога
- Используется нейросеть для обеспечения естественного общения с клиентами.
- Ассистент способен понимать контекст вопросов и давать релевантные ответы.
- Поддержка сложных диалоговых сценариев, включая уточнение информации и корректировку запросов.

### 2. Ответы на вопросы по базе знаний
- База знаний содержит информацию о:
  - Услугах салона (виды окон, материалы, цены).
  - Процессе установки окон.
  - Гарантиях и условиях обслуживания.
  - Часто задаваемых вопросах (FAQ).
- Ассистент оперативно извлекает информацию из базы знаний и предоставляет точные ответы.

### 3. Запись клиента на встречу
- Возможность записаться на консультацию или замер окон через диалог с ассистентом.
- Автоматическая проверка доступности времени в календаре.
- Добавление встречи в Google Календарь с указанием всех необходимых деталей (дата, время, контактные данные клиента).

---

## Используемые сервисы

### 1. **Qwen**

- Настройка диалоговых сценариев и логики взаимодействия с пользователями.

### 3. **GitHub**
- Хостинг кода и документации проекта.
- Используется для совместной разработки и управления версиями.

## Техническая реализация

### Архитектура системы
1. **Фронтенд**: Интерфейс для взаимодействия с клиентами (чат на сайте или мессенджеры).
2. **Бэкенд**: Логика обработки запросов, интеграция с Qwen и Suvvy.ai.
3. **База данных**: Хранение базы знаний и информации о клиентах.
4. **Интеграции**:
   - Google Календарь для записи встреч.
   - API Qwen для генерации ответов.
   - Suvvy.ai для управления диалогами.
   - GitHub для хостинга кода и управления версиями.

### Пример использования
1. Клиент заходит на сайт салона "Белар" и открывает чат с ассистентом.
2. Клиент спрашивает: "Какие виды окон вы предлагаете?"  
   Ассистент обращается к базе знаний и предоставляет информацию о доступных типах окон.
3. Клиент интересуется: "Когда можно записаться на замер?"  
   Ассистент проверяет доступное время в календаре и предлагает варианты.
4. После подтверждения времени ассистент добавляет встречу в Google Календарь и отправляет клиенту подтверждение.
